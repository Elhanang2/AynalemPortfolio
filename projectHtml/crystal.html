Crystal Collector

Simple click game there are four crystals with random values and target random number. when we click each crystal we can see the values. once random values set to crystal it will not change until new game starts. so by clicking the crystals we can reach target value. this game is helpful for childrens skill to add numbers and memorization. crystal numbers are not visible but when we click and numbers added we can get from result. In addition it will count how many time winning and lossing the game.The app is mobile Responsive app

Technology used: HTML, CSS, javaScript and jQuery
